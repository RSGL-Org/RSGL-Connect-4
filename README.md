# RSGL-Connect-4


# Building linux
    git clone https://github.com/RSGL-Org/RSGL-Connect-4.git
    
    cd RSGL-Connect-4
    
    make 

    ./Connect4

# Bulding 3ds (3ds support by SaCode)
    Install devkitpro / 3ds-dev (more info on https://devkitpro.org/wiki/Main_Page)
    
    git clone https://github.com/RSGL-Org/RSGL-Connect-4.git

    cd RSGL-Connect-4

    make

    The 3ds binaries are in ./3ds
